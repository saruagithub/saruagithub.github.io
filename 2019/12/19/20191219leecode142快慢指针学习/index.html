2 hash法与快慢指针法

快慢指针法有意思的推导：

x：link起点到入环点距离

y: 入环点到相遇点距离

c：circle的长度

相遇时候，慢指针走了x+n1 c + y （n1假设走了n1圈），快指针走了2倍(x+ n1c+y)

快指针比慢指针多走的路程一定是环长度的整数倍，有2(x+ n1c+y) - (x+ n1c+y) = n2 c

所以又x + y = (n2 - n1) c

可以相遇时快指针从起点再走（1倍速），慢指针也走，则相遇点就是入环点。
